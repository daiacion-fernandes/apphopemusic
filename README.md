# Hope Music App - OpenCart Application

Uma aplicação para integração de música em lojas OpenCart.

## 📋 Descrição

Hope Music App é um aplicativo desenvolvido para OpenCart que permite gerenciar e integrar conteúdo musical em sua loja online.

## ✨ Características

- Gerenciamento de catálogo musical
- Integração fácil com OpenCart
- Interface intuitiva
- Suporte a múltiplas versões do OpenCart

## 📦 Instalação

### Pré-requisitos
- OpenCart 3.x ou superior
- PHP 7.4 ou superior
- Acesso ao painel de administração da loja

### Passos de Instalação

1. **Download do Arquivo**
   - Baixe o arquivo `.ocmod.zip` da seção [Releases](../../releases)

2. **Upload via Painel Admin**
   - Acesse o painel administrativo do OpenCart
   - Vá para `Extensões > Instalador`
   - Faça upload do arquivo `.ocmod.zip`
   - Clique em "Continuar"

3. **Ativar a Extensão**
   - Vá para `Extensões > Modificações`
   - Procure por "Hope Music App"
   - Clique no ícone verde para ativar

4. **Limpar Cache**
   - Sistema > Cache
   - Clique em "Limpar" para aplicar as alterações

## 🚀 Como Usar

1. Acesse o painel de administração
2. Navegue até `Extensões > Hope Music App`
3. Configure conforme necessário
4. Salve as configurações
